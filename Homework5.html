The following code marks these points in the scatter plot. From the plots it is clear that (a) observation 19 has an extreme x7 value, (b) observation has an unusual y value when plotted against x6, (c) observations 10, 26, 27 are quite unusual in both response and regressor variable values when y is plotted against x7.

cols <- character(nrow(dat))

Now let us take a look at the residual plot for verifying 0 mean and constant variance assumption and the normal q-q plot. The following code creates those plots and marks the flagged observations in red color. We note that, the magnitude of residual for observations 17 and 26 are noticeably high and these points are far away from the q-q line.

par(mfrow = c(1, 2))
## Check for zero mean and constant variance of random error
plot(mlr$fitted.values, mlr$residuals, 
     main = "Check for 0 mean and constant var \n  Residual vs. fitted value")
for (i in 1:length(pts)){
 points(mlr$fitted.values[pts[i]], mlr$residuals[pts[i]], col = "red", pch = 19)
 text(mlr$fitted.values[pts[i]], (mlr$residuals[pts[i]]-1), pts[i], col = "blue")

We note that the estimated regression coefficients of the new model are noticeably different from the previous one. The associate standard errors are also smaller. This is not surprising because both observations 17 and 26 are identified as influential point by the covratio statistic which provides insight about the precision of estimation. We note that the residual standard error is smaller for the new model which is a good thing because smaller MSE (residual standard error square) results in narrower prediction and confidence intervals. Finally we may create the residual plot and q-q plot for the new model as below. We note that the residual plot has improved and the normal q-q plot suggests that error terms follow normal distribution.

par(mfrow = c(1, 2))
## Check for zero mean and constant variance of random error
plot(mlr2$fitted.values, mlr2$residuals, 
     main = "Check for 0 mean and constant var \n  Residual vs. fitted value \n after removing observations 17 and 26")
abline(h=0)

## Check for normality of random error
qqnorm(mlr2$residuals, main = "Normal Q-Q Plot \n after removing observations 17 and 26")
qqline(mlr2$residuals)

We conclude that observations 17 and 26 has a noticeable impact on the regression model where y is regressed on x6 and X7. Whether we can safely remove them or not from the analysis depends on the context of the particular business problem.
